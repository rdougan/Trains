timeInWordsFromTimeInterval:includingSeconds:

Returns a string representing a time interval in words.

+ (NSString *)timeInWordsFromTimeInterval:(NSTimeInterval)intervalInSeconds includingSeconds:(BOOL)includeSeconds

Parameters

intervalInSeconds

The time interval to convert to a string

includeSeconds

YES if seconds should be included. NO if they should not.

Return Value

A string representing the time interval in words

Discussion

Returns a string representing a time interval in words.

The strings produced by this method will be similar to produced by ActiveSupport’s time_ago_in_words helper method.

Return Value

A string representing the time interval from now in words

Discussion

Returns a string representing the time interval from now in words (including seconds).

The strings produced by this method will be similar to produced by Twitter for iPhone or Tweetbot in the top right of the tweet cells.

Internally, this does not use timeInWordsFromTimeInterval:includingSeconds:.
