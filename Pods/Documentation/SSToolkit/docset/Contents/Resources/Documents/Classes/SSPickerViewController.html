SSPickerViewController Class Reference

Inherits from UITableViewController
Declared in SSPickerViewController.h

Overview

This is an abstract class for displaying a UITableView with a list of items for the user to choose similar to Settings.app.

A subclass should override the - (void)loadObjects and - (NSString *)cellTextForObject:(id)anObject methods to customize this class.

A subclass can optionally override - (void)cellImageForKey:(id)key to show an image in the cell.
