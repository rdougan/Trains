alpha

The receiver’s alpha value. (read-only)

@property (nonatomic, assign, readonly) CGFloat alpha

Discussion

The receiver’s alpha value. (read-only)

The value of this property is a floating-point number in the range 0.0 to 1.0, where 0.0 represents totally transparent and 1.0 represents totally opaque.

colorWithHex:

Creates and returns an UIColor object containing a given value.

+ (UIColor *)colorWithHex:(NSString *)hex

Parameters

hex

The value for the new color. The # sign is not required.

Return Value

An UIColor object containing a value.

You can specify hex values in the following formats: rgb, rrggbb, or rrggbbaa.

The default alpha value is 1.0.

Discussion

hexValue

Returns the receiver’s value as a hex string.

- (NSString *)hexValue

Return Value

The receiver’s value as a hex string.

The value will be nil if the color is in a color space other than Grayscale or RGB. The # sign is omitted. Alpha will be omitted.

Discussion

Returns the receiver’s value as a hex string.

hexValueWithAlpha:

Returns the receiver’s value as a hex string.

- (NSString *)hexValueWithAlpha:(BOOL)includeAlpha

Parameters

includeAlpha

YES if alpha should be included. NO if it should not.

Return Value

The receiver’s value as a hex string.

The value will be nil if the color is in a color space other than Grayscale or RGB. The # sign is omitted. Alpha is included if includeAlpha is YES.
