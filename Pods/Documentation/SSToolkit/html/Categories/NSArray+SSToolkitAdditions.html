Return Value

deepMutableCopy

Returns the object returned by mutableCopyWithZone: where the zone is nil.

- (NSMutableArray *)deepMutableCopy

Return Value

The object returned by the NSMutableCopying protocol method mutableCopyWithZone:, where the zone is nil.

Discussion

Returns the object returned by mutableCopyWithZone: where the zone is nil.

This differs from mutableCopy in that it makes any contained NSArray objects or NSDictionary objects mutable as well. The returned dictionary follows standard memory management conventions for copied objects. You are responsible for releasing it.
