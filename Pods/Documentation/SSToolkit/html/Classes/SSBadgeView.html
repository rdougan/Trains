Inherits from UIView
Declared in SSBadgeView.h

Overview

Badge view.

Provides the accesory view for a SSBadgeTableViewCell, available through the badgeView property.

Acts very much like the badges in Mail.app, with the key difference being that Apple uses images and SSBadgeView is rendered with Core Graphics for improved scrolling performance (although images are supported). This also allows for more flexible resizing.
