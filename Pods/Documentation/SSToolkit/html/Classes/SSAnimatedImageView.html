Inherits from UIImageView

Overview

Animated image view.

SSAnimatedImageView is a simple subclass of UIImageView that provides more advanced control of image animations. The animation is backed by a CAKeyframeAnimation.

This class is a work in progress.

Start an image animation.

- (void)startImageAnimation:(NSString *)animationName images:(NSArray *)images delegate:(id<SSAnimatedImageViewDelegate>)delegate

Parameters

animationName

The named of the animation.

images

An array of UIImage objects that will be animated.

delegate

Discussion

Start an image animation.

The duration is set to the appropriate length for playing the images at 30fps.
