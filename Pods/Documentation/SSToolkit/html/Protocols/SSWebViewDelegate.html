SSWebViewDelegate Protocol Reference

Conforms to NSObject
Declared in SSWebView.h

Overview

The SSWebViewDelegate protocol defines methods that a delegate of a SSWebView object can optionally implement to intervene when web content is loaded.

Warning: Important: Before releasing an instance of SSWebView for which you have set a delegate, you must first set the SSWebView delegate property to nil before disposing of the SSWebView instance. This can be done, for example, in the dealloc method where you dispose of the SSWebView.
